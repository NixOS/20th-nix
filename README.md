## Development

1. [Install devenv](https://devenv.sh/getting-started/)

2. Start the processes:

```
      $ devenv up
```

3. Open http://localhost:5100/

### Syncing meetups

Assuming you have `NOTION_API_KEY` set:

   $ devenv shell refresh-meetups
